# Sicherheit

- Login und Passwortverwaltung über Supabase Auth
- keine Passwortspeicherung in der App
- jeder Datensatz ist an das angemeldete Konto gebunden
- Row-Level-Security auf allen Tabellen
- kein Service-Role-Key im Browser oder Repository
- Zugangsdaten über Umgebungsvariablen
- HTTPS über Supabase und GitHub Pages
